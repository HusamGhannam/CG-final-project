# Rolling Ball 3D Game
A custom 3D OpenGL game built from scratch using C++, GLFW, and GLEW.

## Features
* Custom Math Library (No GLM used)
* Procedural 3D track generation
* Sphere-to-AABB collision detection
* Interactive 2D UI for scoring

## How to Build
1. Open the project in Visual Studio.
2. Ensure you have GLFW and GLEW linked in your project properties.
3. Set the build architecture to x86.
4. Build and Run.

## Controls
1. A/D for moving left and right
2. Space to jump
